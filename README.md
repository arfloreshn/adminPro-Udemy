# AdminPro-Udemy

Este proyecto esta creado en Angular 8, su finalidad es que alumno aprenda a utilizar el backEnd y FrontEnd con MongoDB
para installar front-end en su computadora despues de bajar el proyecto.

Ejecute el comando
npm install

# Requisitos
   
  PrimeNG 8 
  Angular 8.3.6
  Node version 10
  Express
